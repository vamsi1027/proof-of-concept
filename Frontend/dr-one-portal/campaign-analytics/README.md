# React Micro App Template

## Quick start

1. Download this project and paste it in [DR-One-Portal](https://github.com/Digita1Reef/dr-one-portal "DR-One-Portal")  project and set the feature name in it;

2. Change the main file name to the same project name on `src/dr-one-[project-name].tsx`

3. Change project global variables in `webpack.config.js`, `package.json` and `userHooks.tsx`;

4. Import the project at index.ejs on root folder;

5. Install the dependencies:

`yarn`

6. Running:

`yarn start`
